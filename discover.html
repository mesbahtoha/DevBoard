Question-1: What are the different ways to select an element in the DOM?

We can select DOM elements using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. Among these, querySelector and querySelectorAll are most flexible as they support any CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns or sets HTML content inside an element, including tags. innerText and textContent return only text, but innerText respects styles (like hidden elements), while textContent shows all text as-is.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements. It works by using event bubbling, allowing efficient event handling even for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and bubbles up to its parent elements. This means event handlers on parent elements can also respond to events triggered on their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements using document.createElement(), add them with appendChild() or append(), and remove them using removeChild() or the element’s remove() method. Example: const div = document.createElement("div"); parent.appendChild(div); div.remove();
